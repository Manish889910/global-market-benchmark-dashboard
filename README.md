# 🌍 Global Market Benchmark Dashboard

This interactive Streamlit dashboard analyzes global e-commerce sales data 
to evaluate international market expansion potential.

## 📊 Key Features
- Revenue & KPI tracking
- Category & Country benchmarking
- Monthly trend analysis
- Currency conversion (USD base)
- Strategic Insight section
- Methodological transparency note

## 📌 Methodology
The dataset represents global e-commerce sales excluding Russia.
Market entry recommendations are derived from comparative global demand
patterns and macro-environmental analysis.

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- SQL (Data Analysis)

## 🚀 Deployment
Deployed using Streamlit Community Cloud.

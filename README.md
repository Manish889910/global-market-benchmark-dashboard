# 🌍 Global Market Benchmark Dashboard

An interactive analytics dashboard designed to evaluate international market expansion feasibility using global e-commerce benchmarking and macro-environmental assessment.

---

## 📊 Key Features
- 📈 Revenue & KPI tracking
- 🌎 Country & category benchmarking
- 📅 Monthly sales trend analysis
- 💱 Dynamic currency conversion (USD base reference)
- 🧠 Strategic insight summary
- 📌 Transparent methodology disclosure

---

## 📌 Methodology
The dataset represents global e-commerce sales **excluding Russia**.  
Therefore, Russian market entry recommendations are derived from comparative global demand trends and macro-environmental (PESTEL) analysis rather than direct Russian transactional data.
